# QDB-2.0

Rewrite of the SSE QDB using rails 4.1 and some other technologies not yet decided upon

## Technologies
* Rails 4.1
* Ruby 2.0
* Slim
* Polymer

## Dependencies
* Ruby 1.9.3+
* Node and NPM
* Bower

## Development

1. `bundle install`
2. `rake db:create`
3. `rake db:migrate`
4. `rake bower:install`
5. `rails server`
